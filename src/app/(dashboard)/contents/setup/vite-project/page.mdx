> 📌 **この記事で学ぶこと**
>
> - Vite とは何か
> - React プロジェクトの作成手順
> - 開発サーバーの起動方法
> - ブラウザでの確認

## Vite とは？

**Vite**（ヴィート）は、Web 開発のためのビルドツールです。
フランス語で「速い」という意味で、その名の通り**とにかく速い**のが特徴。

Vite を使うと：

- React プロジェクトを簡単に作成できる
- ファイルを保存するとすぐにブラウザに反映される
- 開発がとても快適になる

React の公式も、新規プロジェクトには Vite の使用を推奨しています。

## プロジェクトを作成しよう

いよいよ、React プロジェクトを作成します！

### 1. 作業フォルダを作成

まず、コースで作るプロジェクトをまとめる**専用フォルダ**を作りましょう。
デスクトップに `webex` フォルダを作って、そこに全てのプロジェクトを作成していきます。

```bash
# デスクトップに移動
cd ~/Desktop

# webex フォルダを作成（コース専用）
mkdir webex

# webex フォルダに移動
cd webex
```

> 💡 **なぜ専用フォルダを作るの？**
>
> - コースで作ったファイルが一箇所にまとまって管理しやすい
> - 他のファイルと混ざらない
> - 後から見返しやすい
>
> **今後作成するプロジェクトは、すべてこの `webex` フォルダの中に作りましょう！**

### 2. Vite で React プロジェクトを作成

以下のコマンドを実行します：

```bash
npm create vite@latest my-app -- --template react
```

> 💡 **コマンドの意味**
>
> - `npm create vite@latest`: 最新の Vite を使ってプロジェクトを作成
> - `my-app`: プロジェクト名（好きな名前で OK）
> - `--template react`: React のテンプレートを使う

### 3. プロジェクトフォルダに移動

```bash
cd my-app
```

### 4. パッケージをインストール

```bash
npm install
```

これで、React に必要なパッケージが全てインストールされます。
少し時間がかかることがあります。


## 開発サーバーについて

### 開発サーバーとは？

`npm run dev` を実行すると、開発用のサーバーが起動します。

このサーバーは：

- ファイルを保存すると、自動でブラウザに反映される（ホットリロード）
- エラーがあれば、ブラウザに表示してくれる
- 開発中だけ使う（本番公開用ではない）

### サーバーを停止するには

ターミナルで `Ctrl + C` を押すと、開発サーバーが停止します。

```
^C
```

再度起動したい場合は、もう一度 `npm run dev` を実行します。

## VSCode でプロジェクトを開く

作成したプロジェクトを VSCode で開いてみましょう。

### 方法 1: ターミナルから

```bash
code .
```

> 💡 このコマンドが動かない場合は、VSCode のコマンドパレット（`Cmd + Shift + P`）で「Shell Command: Install 'code' command in PATH」を実行してください。

### 方法 2: VSCode から

1. VSCode を開く
2. ファイル → フォルダを開く
3. `my-app` フォルダを選択

## ファイルを編集してみよう

試しに、表示を変更してみましょう。

1. VSCode で `src/App.jsx` を開く

2. 「Vite + React」と書かれている部分を探す

3. 好きな文字に変更して保存（`Cmd + S` / `Ctrl + S`）

4. ブラウザを見ると、**自動で更新**されている！

<!--
📸 画像メモ: App.jsx を編集している様子
- 「Hello World!」などに変更
- ブラウザで変更が反映されている
-->

これが「**ホットリロード**」です。
ファイルを保存するだけで、すぐに結果を確認できます。

## コマンドまとめ

| コマンド                                            | 説明                       |
| --------------------------------------------------- | -------------------------- |
| `npm create vite@latest my-app -- --template react` | プロジェクト作成           |
| `cd my-app`                                         | プロジェクトフォルダに移動 |
| `npm install`                                       | パッケージをインストール   |
| `npm run dev`                                       | 開発サーバーを起動         |
| `Ctrl + C`                                          | 開発サーバーを停止         |

## まとめ

- **Vite** は高速な開発ツール
- `npm create vite` で React プロジェクトを簡単に作成できる
- `npm run dev` で開発サーバーを起動
- ファイルを保存すると**自動でブラウザに反映**される
