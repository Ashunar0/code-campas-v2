## HTML とは？

**HTML**（HyperText Markup Language）は、Web ページの**構造**を作るための言語です。

HTML を使って「これは見出し」「これは段落」「これは画像」といった情報を**マークアップ**（印をつける）することで、ブラウザがページを正しく表示できるようになります。

## HTML の基本構造

HTML ファイルの基本構造は以下のようになっています：

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>ページのタイトル</title>
  </head>
  <body>
    {/* ここにページの内容を書く */}
    <h1>Hello World!</h1>
    <p>これは段落です。</p>
  </body>
</html>
```

### 各部分の役割

| 要素              | 説明                                         |
| ----------------- | -------------------------------------------- |
| `<!DOCTYPE html>` | HTML5 であることを宣言                       |
| `<html>`          | HTML 全体を囲む                              |
| `<head>`          | ページの設定情報（ブラウザには表示されない） |
| `<body>`          | ページの本文（ブラウザに表示される）         |

> 💡 React では、この「雛形」部分は自動で用意されるので、`<body>` の中身だけ書くイメージです。

## 基本的なタグ

### 見出し（h1 〜 h6）

ページのタイトルや章タイトルに使います。
h1 が最も大きく、h6 が最も小さいです。

```html
<h1>大見出し</h1>
<h2>中見出し</h2>
<h3>小見出し</h3>
```

📸 画像メモ: h1〜h3 の表示例
*/}

### 段落（p）

文章のかたまりを表します。

```html
<p>これは段落です。段落は文章のまとまりを表します。</p>
<p>新しい段落は、新しい p タグで囲います。</p>
```

### ブロック要素（div）

汎用的なコンテナ（入れ物）です。
要素をグループ化するときに使います。

```html
<div>
  <h2>セクションのタイトル</h2>
  <p>セクションの内容</p>
</div>
```

### リンク（a）

他のページへのリンクを作ります。

```html
<a href="https://google.com">Google へ移動</a>
<a href="./about.html">About ページへ</a>
```

- `href`：リンク先の URL を指定

### 画像（img）

画像を表示します。

```html
<img src="image.jpg" alt="画像の説明" />
```

- `src`：画像ファイルのパス
- `alt`：画像が表示されないときの代替テキスト

> ⚠️ `<img>` タグは閉じタグがありません（自己終了タグ）

### ボタン（button）

クリックできるボタンを作ります。

```html
<button>クリック</button>
```

### リスト（ul, ol, li）

箇条書きを作ります。

**順序なしリスト（ul）**

```html
<ul>
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul>
```

**順序ありリスト（ol）**

```html
<ol>
  <li>手順1</li>
  <li>手順2</li>
  <li>手順3</li>
</ol>
```

{/*
📸 画像メモ: ul と ol の表示の違い
- ul: 黒丸（・）
- ol: 番号（1. 2. 3.）
*/}

### フォーム関連（input, form）

ユーザーからの入力を受け付けます。

```html
<form>
  <input type="text" placeholder="名前を入力" />
  <input type="email" placeholder="メールアドレス" />
  <input type="password" placeholder="パスワード" />
  <button type="submit">送信</button>
</form>
```

| type       | 説明                         |
| ---------- | ---------------------------- |
| `text`     | 通常のテキスト入力           |
| `email`    | メールアドレス入力           |
| `password` | パスワード入力（●●● で表示） |
| `number`   | 数値入力                     |
| `checkbox` | チェックボックス             |

## 属性とは

タグに追加情報を付けるものです。

```html
<タグ名 属性名="値">内容</タグ名>
```

よく使う属性：

| 属性    | 説明                         | 使用例                    |
| ------- | ---------------------------- | ------------------------- |
| `class` | CSS でスタイルを適用するため | `<div class="container">` |
| `id`    | 要素を一意に識別             | `<div id="header">`       |
| `href`  | リンク先                     | `<a href="...">`          |
| `src`   | ファイルのパス               | `<img src="...">`         |
| `alt`   | 代替テキスト                 | `<img alt="...">`         |

## セマンティック HTML

HTML5 では、より意味のある（セマンティックな）タグが追加されました。

```html
<header>ヘッダー</header>
<nav>ナビゲーション</nav>
<main>メインコンテンツ</main>
<section>セクション</section>
<article>記事</article>
<aside>サイドバー</aside>
<footer>フッター</footer>
```

これらを使うと：

- コードが読みやすくなる
- SEO（検索エンジン最適化）に良い
- アクセシビリティが向上

## 試してみよう

VSCode で `index.html` を作成し、以下のコードを書いてみましょう：

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>はじめての HTML</title>
  </head>
  <body>
    <header>
      <h1>My First Page</h1>
    </header>

    <main>
      <h2>自己紹介</h2>
      <p>こんにちは！プログラミングを学んでいます。</p>

      <h2>好きなもの</h2>
      <ul>
        <li>コーヒー</li>
        <li>音楽</li>
        <li>プログラミング</li>
      </ul>

      <h2>リンク</h2>
      <a href="https://google.com">Google</a>
    </main>

    <footer>
      <p>&copy; 2024 My Name</p>
    </footer>
  </body>
</html>
```

ファイルをブラウザで開いて、表示を確認してみましょう！

> 💡 **ブラウザで開く方法（Live Server）**
>
> 1. VSCode で `index.html` を開いた状態で
> 2. ファイル名を**右クリック**
> 3. 「**Open with Live Server**」を選択
>
> ブラウザが自動で開いてページが表示されます。
> ファイルを編集して保存すると、自動でブラウザも更新されます！

## まとめ

- **HTML** は Web ページの構造を作る言語
- **タグ**で内容をマークアップする
- よく使うタグ：`div`, `p`, `h1`〜`h6`, `a`, `img`, `ul`, `li`
- **属性**でタグに追加情報を付ける（class, id, href, src など）
- **セマンティックタグ**（header, main, footer など）で意味のある構造に
