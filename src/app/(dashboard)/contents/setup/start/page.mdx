## この章について

Web アプリを作り始める前に、まず「開発環境」を整える必要があります。

料理に例えるなら、レシピ（コード）を書く前に、キッチン（開発環境）を整えるようなもの。包丁やまな板がないと料理ができないように、開発ツールがないとプログラムは書けません。

この章では、Web アプリを作るために必要なツールを一通りインストールしていきます。

## 環境構築の重要性

「環境構築」は、プログラミング初心者が**最初に挫折しやすいポイント**です。

なぜなら：

- エラーメッセージが英語で読めない
- 何をやっているのかわからないまま進んでしまう
- 人によって PC の状態が違うので、同じ手順でもエラーが出ることがある

でも安心してください。**つまづく部分があったら、メンターと一緒に解決しましょう！**

## 困ったときは

環境構築でエラーが出た場合は、以下の順番で対処してみてください：

### 1. エラーメッセージを読む

英語でも、Google 翻訳を使えば大丈夫！
エラーメッセージには「何が問題か」が書いてあることが多いです。

### 2. 再起動してみる

ターミナルを閉じて開き直す、PC を再起動する。
意外とこれで直ることも多いです。

### 3. エラーメッセージで検索する

エラーメッセージをそのままコピーして Google 検索。
同じエラーで困った人の解決策が見つかることがあります。

### 4. メンターに聞く

それでもダメなら、遠慮せずメンターに聞きましょう！
スクリーンショットを撮って共有すると、解決が早くなります。
