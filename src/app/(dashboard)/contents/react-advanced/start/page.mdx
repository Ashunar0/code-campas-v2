## 学習のコツ

### 手を動かす

React 応用は「書いて、動かして」学ぶのが一番です。
コードをコピペするだけでなく、**自分で書いて試す**ことを意識しましょう。

### 基礎を振り返る

この章は React 基礎章の知識が前提です。
つまずいたら、基礎章に戻って復習しましょう。

> 💡 特に **State** と **Props** の理解は必須です。
> 「State が変わると再レンダリングされる」ことを意識しましょう。

### 任意の記事は後回しでも OK

「（任意）」と書かれた記事は、時間に余裕があるときに読めば OK。
まずは必須の記事（⭐ マーク付き）を優先しましょう。

### わからなくても進む

すべてを完璧に理解してから次へ進む必要はありません。
**実際に動かして**みることで、後から理解が追いつくこともあります。

> 💡 **エラーが出たら、まずエラーメッセージを読みましょう。**
> React のエラーメッセージは具体的なので、ヒントが得られます。
