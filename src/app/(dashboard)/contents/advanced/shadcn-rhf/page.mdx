## Zod とは

型安全なバリデーションライブラリです。

```bash
npm install zod @hookform/resolvers
```

---

## スキーマの定義

```typescript
import { z } from "zod";

const formSchema = z.object({
  email: z.string().email("正しいメールアドレスを入力"),
  password: z.string().min(6, "6文字以上で入力"),
});

type FormData = z.infer<typeof formSchema>;
```

---

## フォームの実装

```typescript
import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";

const form = useForm<FormData>({
  resolver: zodResolver(formSchema),
  defaultValues: {
    email: "",
    password: "",
  },
});

const onSubmit = (data: FormData) => {
  console.log(data);
};
```

---

## shadcn/ui の Form と連携

```tsx
<Form {...form}>
  <form onSubmit={form.handleSubmit(onSubmit)}>
    <FormField
      control={form.control}
      name="email"
      render={({ field }) => (
        <FormItem>
          <FormLabel>メール</FormLabel>
          <FormControl>
            <Input {...field} />
          </FormControl>
          <FormMessage />
        </FormItem>
      )}
    />
    <Button type="submit">送信</Button>
  </form>
</Form>
```

---

## まとめ

- **Zod** は型安全なバリデーションライブラリ
- `z.object()` でスキーマを定義し、`z.infer<>` で型を取得
- **React Hook Form** と **Zod** を組み合わせてフォームを作成
- shadcn/ui の Form コンポーネントで美しいフォームに
