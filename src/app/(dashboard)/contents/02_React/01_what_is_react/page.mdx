## 2. React とは？

React は、ユーザーインターフェース（UI）を作るための JavaScript ライブラリです。
Facebook（現 Meta）が開発して、今では Google、Netflix、Airbnb など、世界中のサービスで使われています。

#### React が人気な理由

- コンポーネント指向：部品（コンポーネント）を組み合わせてアプリを作れるから、見通しが良くて再利用もしやすい！

- 仮想 DOM：HTML を直接いじらず、効率よく画面更新できる！サクサク動く！

- 状態管理がカンタン：useState を使って「今の状態」を画面に反映できる！

#### どんな時に使うか

- <strong>SNSやチャットアプリ</strong>のような動的に画面が変わるもの

- <strong>ボタンを押すとデータが変わる</strong>ようなインタラクティブなUI構築

- 管理画面やダッシュボードなど、<strong>パーツを分けて再利用したいアプリ</strong>

たとえば、こんな感じで

```jsx
const hello = () => {
  return (
    <div>
      <h1>Hello!</h1>
    </div>
  );
};
```

JavaScript の中に直接 HTML を書くことができちゃいます！

つまり React は、<strong>「JavaScript を使ってより簡単に、より高度な Web アプリを作れるツール」</strong>ってわけです!!
このあと、実際に自分で触って体感していきましょう 🙌

## 3. React プロジェクトの作成

それでは、早速 React プロジェクトを作って、軽く触ってみましょう！

##### ① プロジェクトフォルダを作りたい場所に移動して、以下を実行 👇

```sh
npm create vite@latest my-react-app -- --template react
```

##### ② 質問されたらそのまま Enter で OK（my-react-app がプロジェクト名）

##### ③ プロジェクトに入る：

```sh
cd my-react-app
```

##### ④ 依存パッケージをインストール：

```sh
npm install
```

※`npm i`でも ok⭕️

##### ⑤ 開発サーバーを起動：

```sh
npm run dev
```

ブラウザで http://localhost:5173 を開けば、React の初期画面が表示されるはず！

![image.png](/images/image01.png)

#### 動作確認

`src/App.jsx`を開いて、以下のように編集しましょう。

```jsx
// src/App.jsx
const App = () => {
  return <div>Hello React!</div>;
};

export default App;
```

ブラウザで確認して、画面に「Hello React!」と表示されていれば OK！
これで、React 開発の第一歩は完了です！🎉

## 4. ディレクトリ構成を理解しよう

Vite で React プロジェクトを作成すると、最初はこんな構成になっています 👇

```sh
my-react-app/
├── public/
├── src/
│   ├── assets/
│   ├── App.jsx
│   ├── App.css
│   ├── main.js
│   └── index.css
├── .gitignore
├── index.html
├── package.json
├── vite.config.js
└── ...
```

ここでは特に大事なファイル・フォルダだけピックアップして解説していきます 🕵️

#### `public/`

静的ファイル（画像やフォントなど）を入れる場所。

#### `src/`（最重要）

ここが React アプリのコードを書く場所！

##### `App.jsx`

- アプリ全体のメインファイル。基本的に使うのはこれ！！
- 自分で作ったコンポーネントをここに組み込んでいく。
- とりあえずここに書くってことがわかっていれば OK

##### `package.json`

- プロジェクトの設定ファイル。
- 依存ライブラリや、npm run dev などのコマンドが書かれてる。
- ライブラリを追加するとここに記述される。

主に使うのは`App.jsx`なので、余裕がなければ他のファイルは気にしなくて大丈夫です！
