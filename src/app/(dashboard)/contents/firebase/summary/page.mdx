### Firestore（データベース）

| 操作   | 関数                        |
| ------ | --------------------------- |
| 取得   | getDocs, getDoc, onSnapshot |
| 追加   | addDoc, setDoc              |
| 更新   | updateDoc                   |
| 削除   | deleteDoc                   |
| クエリ | where, orderBy, limit       |

### Authentication（認証）

| 機能         | 関数                           |
| ------------ | ------------------------------ |
| ユーザー登録 | createUserWithEmailAndPassword |
| ログイン     | signInWithEmailAndPassword     |
| ログアウト   | signOut                        |
| 認証状態監視 | onAuthStateChanged             |

### Storage（ファイル保存）

| 機能         | 関数           |
| ------------ | -------------- |
| アップロード | uploadBytes    |
| URL 取得     | getDownloadURL |
| 削除         | deleteObject   |

### Hosting（デプロイ）

```bash
npm run build
firebase deploy --only hosting
```

## よくある質問

### Q: Firestore と Realtime Database の違いは？

**A**: Firestore が新しく、より高機能です。
新規プロジェクトでは Firestore を推奨。

### Q: セキュリティルールは本番で必須？

**A**: 必須です。
テストモードのまま公開すると、誰でもデータを読み書きできてしまいます。

### Q: 無料枠を超えると課金される？

**A**: はい。ただし、個人の学習用途なら無料枠で十分です。
Firebase Console で使用量を確認できます。
