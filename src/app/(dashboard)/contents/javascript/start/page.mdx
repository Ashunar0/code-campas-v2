## この章について

この章では、Web アプリに「動き」をつけるための **JavaScript** を学びます。

JavaScript を使うと：

- ボタンをクリックしたら何かが起きる
- ユーザーの入力に応じて表示が変わる
- API からデータを取得して表示する

など、**インタラクティブ**な Web サイトが作れるようになります。

## JavaScript の重要性

「React を使うのに、JavaScript をしっかり学ぶ必要があるの？」

答えは **「はい、絶対に必要です」**。

React は **JavaScript のライブラリ**です。
React のコードを書くということは、JavaScript のコードを書くということ。

特に、以下の知識は React で毎日使います：

- **アロー関数**
- **配列操作**（map, filter, find）
- **非同期処理**（async/await）
- **分割代入**

この章でしっかり身につけておきましょう！

## 学習の進め方

### 手を動かす

JavaScript は「書いて、動かして」学ぶのが一番です。
コードをコピペするだけでなく、**自分で書いて試す**ことを意識しましょう。

### Console を活用

ブラウザの開発者ツールにある **Console** でコードを試せます。
小さなコードを試すのに最適です。

### 任意の記事は後回しでも OK

「（任意）」と書かれた記事は、時間に余裕があるときに読めば OK。
まずは必須の記事を優先しましょう。
