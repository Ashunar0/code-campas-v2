## アクセシビリティとは

**アクセシビリティ（a11y）** は、すべての人がウェブサイトを使えるようにすることです。

```
対象：
├── 視覚障害のある人
├── 聴覚障害のある人
├── 運動機能に制限のある人
├── 一時的な障害（怪我など）
└── 高齢者
```

---

## セマンティック HTML

意味のある HTML タグを使います。

```html
{/* ❌ 悪い例 */}
<div class="header">...</div>
<div class="nav">...</div>
<div class="button">クリック</div>

{/* ✅ 良い例 */}
<header>...</header>
<nav>...</nav>
<button>クリック</button>
```

---

## ARIA 属性

```html
{/* ラベルをつける */}
<button aria-label="メニューを開く">☰</button>

{/* 状態を示す */}
<button aria-pressed="true">いいね</button>

{/* 非表示を示す */}
<div aria-hidden="true">装飾用</div>
```

---

## チェックポイント

```
✅ キーボードで操作できるか
✅ スクリーンリーダーで読めるか
✅ コントラストは十分か
✅ フォーカスが見えるか
```

---

## まとめ

- **アクセシビリティ（a11y）** はすべての人が使えるWebを作ること
- **セマンティック HTML**（`<button>`, `<nav>`）を使う
- **ARIA 属性**で追加の情報を伝える
- キーボード操作、コントラスト、フォーカスを確認
