## Node.js とは？

**Node.js** は、JavaScript をパソコン上で動かすためのソフトウェアです。

もともと JavaScript は「ブラウザの中だけ」で動く言語でした。
しかし Node.js の登場によって、JavaScript がパソコン上でも動くようになり、Web アプリの開発に欠かせないツールになりました。

React のプロジェクトを作ったり、開発サーバーを動かしたりするのに Node.js が必要です。

{/* 
📸 画像メモ: Node.js の役割を示す図
- 左側: ブラウザ（Chrome など）で JavaScript が動く
- 右側: パソコン上で JavaScript（Node.js）が動く
- 両方をつなげて「JavaScript がどこでも動く！」というイメージ
*/}

## npm とは？

**npm** は、Node Package Manager の略で、JavaScript のパッケージ（ライブラリ）を管理するツールです。

「パッケージ」とは、誰かが作った便利なプログラムの塊のこと。
npm を使うと、React や Vite などのツールを簡単にインストールできます。

Node.js をインストールすると、npm も一緒に入ります。

## インストール手順

### Mac の場合

#### 方法 1: 公式サイトからダウンロード（簡単）

1. [Node.js 公式サイト](https://nodejs.org/) にアクセス

2. **LTS 版**（推奨版）をダウンロード

{/* 
📸 画像メモ: Node.js 公式サイトのダウンロードページ
- LTS 版のボタンを赤枠で囲む
- 「LTS = 安定版」のような注釈
*/}

3. ダウンロードした `.pkg` ファイルを開いてインストール

4. 画面の指示に従って「続ける」をクリック

{/* 
📸 画像メモ: Mac のインストーラー画面
- 「続ける」ボタンを押していく様子
*/}

#### 方法 2: Homebrew を使う（上級者向け）

Homebrew がインストール済みの場合は、ターミナルで以下を実行：

```bash
brew install node
```

### Windows の場合

1. [Node.js 公式サイト](https://nodejs.org/) にアクセス

2. **LTS 版**（推奨版）をダウンロード

3. ダウンロードした `.msi` ファイルを開いてインストール

4. 画面の指示に従って「Next」をクリック

{/* 
📸 画像メモ: Windows のインストーラー画面
- 基本的には「Next」を押し続ける
- 途中の設定は変更しなくて OK
*/}

5. 「Install」をクリックしてインストール開始

6. 完了したら「Finish」をクリック

## バージョン確認

インストールが完了したら、正しくインストールされたか確認しましょう。

ターミナル（Mac）またはコマンドプロンプト（Windows）を開いて、以下のコマンドを実行します：

### Node.js のバージョン確認

```bash
node -v
```

こんな感じでバージョンが表示されれば OK です：

```bash
v20.10.0
```

（バージョンの数字は異なる場合があります）

### npm のバージョン確認

```bash
npm -v
```

こんな感じでバージョンが表示されれば OK です：

```bash
10.2.3
```

{/* 
📸 画像メモ: ターミナルで node -v と npm -v を実行した結果
- 実際のコマンドと出力結果が見える状態
*/}

## うまくいかない場合

### 「command not found」と表示される

ターミナル/コマンドプロンプトを**一度閉じて、もう一度開いてから**再実行してみてください。

インストール直後は、パスが反映されていないことがあります。

### それでもダメな場合

PC を再起動してから、もう一度試してみてください。

詳しいトラブルシューティングは よくあるエラーと解決策 を参照してください。

## まとめ

- **Node.js** は JavaScript をパソコンで動かすソフトウェア
- **npm** は JavaScript のパッケージを管理するツール
- Node.js をインストールすると npm も一緒に入る
- `node -v` と `npm -v` でバージョンが表示されれば OK
