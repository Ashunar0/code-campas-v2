## Zustand とは

**Zustand** は、シンプルな状態管理ライブラリです。

```bash
npm install zustand
```

---

## 基本的な使い方

```typescript
import { create } from 'zustand';

interface Store {
  count: number;
  increment: () => void;
  decrement: () => void;
}

const useStore = create<Store>((set) => ({
  count: 0,
  increment: () => set((state) => ({ count: state.count + 1 })),
  decrement: () => set((state) => ({ count: state.count - 1 })),
}));
```

---

## コンポーネントで使う

```tsx
function Counter() {
  const { count, increment, decrement } = useStore();

  return (
    <div>
      <p>{count}</p>
      <button onClick={increment}>+</button>
      <button onClick={decrement}>-</button>
    </div>
  );
}
```

---

## 認証状態の管理

```typescript
interface AuthStore {
  user: User | null;
  setUser: (user: User | null) => void;
}

const useAuthStore = create<AuthStore>((set) => ({
  user: null,
  setUser: (user) => set({ user }),
}));
```

---

## まとめ

- **Zustand** はシンプルで軽量な状態管理ライブラリ
- `create<Store>()` でストアを作成し、どのコンポーネントからも使える
- Context API より**ボイラープレートが少ない**
- 認証状態やテーマなど、**グローバルに共有したい状態**に最適
