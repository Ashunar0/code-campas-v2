## 次の章に進む前に

### 確認しておきたいこと

1. **PC を再起動しても開発サーバーが起動できるか**

   - ターミナルで `cd my-app` → `npm run dev`

2. **VSCode でファイルを編集して保存できるか**

   - `App.jsx` を編集 → 保存 → ブラウザに反映

3. **わからないことをメモしているか**
   - 疑問点は早めにメンターに質問しましょう

### よくある質問

**Q: 毎回 `npm run dev` を実行しないといけないの？**

A: はい。PC を再起動したり、ターミナルを閉じると開発サーバーは停止します。
開発を再開するときは、毎回 `npm run dev` を実行してください。

**Q: `my-app` 以外の名前でプロジェクトを作ってもいい？**

A: もちろん！ `npm create vite@latest 好きな名前 -- --template react` で OK です。
ただし、スペースや日本語は避けましょう。

**Q: 設定や拡張機能を後から追加・変更してもいい？**

A: 大丈夫です！必要に応じて拡張機能を追加したり、設定を変更したりしてください。

## メンターへの質問の仕方

わからないことがあれば、遠慮せずメンターに質問しましょう！

質問するときは、以下の情報を伝えると解決が早くなります：

### 伝えるべき情報

1. **何をしようとしたか**

   - 「npm install を実行しようとした」など

2. **どんなエラーが出たか**

   - スクリーンショットがベスト
   - エラーメッセージをコピペでも OK

3. **何を試したか**
   - 「ターミナルを再起動した」など

### スクリーンショットの撮り方

- **Mac**: `Cmd + Shift + 4`（範囲選択）または `Cmd + Shift + 3`（全画面）
- **Windows**: `Windows + Shift + S`（範囲選択）
