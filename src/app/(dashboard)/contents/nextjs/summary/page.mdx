## MVP で Next.js を使うかの判断

### React のみ（Vite）を選ぶ場合

```
├── シンプルな SPA
├── SEO が重要でない
├── Firebase Hosting でデプロイ
├── 学習コストを抑えたい
```

### Next.js を選ぶ場合

```
├── 複数ページのサイト
├── SEO が重要
├── Vercel でデプロイしたい
├── サーバーサイドの処理が必要
```
