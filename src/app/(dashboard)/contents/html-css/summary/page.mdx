## この章で学んだこと

おつかれさまでした！🎉

HTML/CSS 章では、Web ページの基礎を学びました。

### HTML

- **HTML** は Web ページの構造を作る言語
- 基本的なタグ：`div`, `p`, `h1`〜`h6`, `a`, `img`, `ul`, `li`, `input`
- 属性：`class`, `id`, `href`, `src`
- セマンティック HTML：`header`, `main`, `footer`, `section`

### CSS

- **CSS** は Web ページの見た目を整える言語
- セレクタ：タグ、クラス（.）、ID（#）
- よく使うプロパティ：`color`, `background-color`, `margin`, `padding`, `border`

### レイアウト

- **Box Model**：content, padding, border, margin
- **Flexbox**：1 次元のレイアウト（横並び・縦並び）
- **Grid**：2 次元のレイアウト（行 × 列）
- **レスポンシブデザイン**：メディアクエリで画面サイズに対応

### 実践

- ポートフォリオサイトを作成
- Tailwind CSS の導入

## React でどう活かされるか

HTML/CSS の知識は、React 開発で**そのまま使えます**。

### JSX = ほぼ HTML

React では JSX という形式でコードを書きますが、これは HTML とほぼ同じです。

```jsx
// React (JSX)
<div className="container">
  <h1>Hello World</h1>
  <p>Welcome to React</p>
</div>
```

違いは：

- `class` → `className`
- `onclick` → `onClick`（キャメルケース）

### CSS はそのまま使える

React でも CSS はそのまま使えます。

- 通常の CSS ファイル
- CSS Modules
- **Tailwind CSS**（おすすめ！）

## 完璧じゃなくて OK

HTML/CSS のタグやプロパティは膨大です。
**全部覚える必要はありません**。

覚えておくべきこと：

- よく使うタグ・プロパティは自然と覚える
- わからないことは調べれば OK
- 実際に作りながら学ぶのが一番
