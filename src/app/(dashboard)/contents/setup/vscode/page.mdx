## VSCode とは？

**VSCode**（Visual Studio Code）は、Microsoft が開発した無料のコードエディタです。
世界中のプログラマに最も使われているエディタの一つで、React 開発にも最適です。

特徴：

- 無料で使える
- 拡張機能が豊富
- 軽くて動作が速い
- 日本語に対応

## インストール

1. [VSCode 公式サイト](https://code.visualstudio.com/) にアクセス

2. 「Download」ボタンをクリック（OS を自動判別してくれます）

<!-- 
📸 画像メモ: VSCode 公式サイトのダウンロードページ
- ダウンロードボタンを赤枠で囲む
-->

3. ダウンロードしたファイルを開いてインストール

**Mac の場合：**
- ダウンロードした `.zip` を解凍
- `Visual Studio Code.app` をアプリケーションフォルダにドラッグ

**Windows の場合：**
- ダウンロードした `.exe` を実行
- 画面の指示に従ってインストール

## 日本語化

VSCode はデフォルトで英語表示なので、日本語化しましょう。

1. VSCode を開く

2. 左側のサイドバーにある **拡張機能アイコン**（四角が4つのマーク）をクリック

<!-- 
📸 画像メモ: 拡張機能アイコンの場所
- サイドバーの拡張機能アイコンを赤枠で囲む
-->

3. 検索バーに `Japanese` と入力

4. 「Japanese Language Pack for Visual Studio Code」を見つけて「**Install**」をクリック

<!-- 
📸 画像メモ: Japanese Language Pack の検索結果
- 正しい拡張機能を赤枠で囲む
- Install ボタンの場所を示す
-->

5. インストール後、右下に表示される「**Restart**」をクリック

6. 再起動後、メニューが日本語になっていれば OK！

## おすすめの拡張機能

以下の拡張機能をインストールしておくと、開発がとても快適になります。

### 必須レベル

| 拡張機能名 | 説明 |
| --------- | ---- |
| **Prettier - Code formatter** | コードを自動で整形してくれる |
| **ES7+ React/Redux/React-Native snippets** | React のコードを素早く書けるショートカット |
| **Auto Rename Tag** | HTML タグを編集すると、閉じタグも自動で変わる |
| **Live Server** | HTML ファイルをブラウザでプレビューできる |

### あると便利

| 拡張機能名 | 説明 |
| --------- | ---- |
| **Tailwind CSS IntelliSense** | Tailwind CSS の補完が効くようになる |
| **Material Icon Theme** | ファイルのアイコンが見やすくなる |
| **Error Lens** | エラーをコードの横に表示してくれる |

### インストール方法

1. 拡張機能アイコンをクリック
2. 検索バーに拡張機能名を入力
3. 「Install」をクリック

これを、上記の拡張機能に対して繰り返します。

## Prettier の設定

**Prettier** は、コードを自動で整形してくれる拡張機能です。
保存するたびに自動で整形されるように設定しましょう。

1. `Cmd + ,`（Mac）または `Ctrl + ,`（Windows）で設定を開く

2. 検索バーに `format on save` と入力

3. 「**Editor: Format On Save**」にチェックを入れる

<!-- 
📸 画像メモ: Format On Save の設定画面
- 設定画面で「Format On Save」を検索した状態
- チェックボックスにチェックが入っている状態
-->

4. 検索バーに `default formatter` と入力

5. 「**Editor: Default Formatter**」で「Prettier - Code formatter」を選択

これで、ファイルを保存するたびにコードが自動で整形されます！

## 基本的なショートカットキー

開発を効率化するために、よく使うショートカットキーを覚えておきましょう。

### ファイル操作

| 操作 | Mac | Windows |
| ---- | --- | ------- |
| 保存 | `Cmd + S` | `Ctrl + S` |
| 新規ファイル | `Cmd + N` | `Ctrl + N` |
| ファイルを開く | `Cmd + O` | `Ctrl + O` |
| ファイル検索 | `Cmd + P` | `Ctrl + P` |

### 編集

| 操作 | Mac | Windows |
| ---- | --- | ------- |
| コピー | `Cmd + C` | `Ctrl + C` |
| 貼り付け | `Cmd + V` | `Ctrl + V` |
| 元に戻す | `Cmd + Z` | `Ctrl + Z` |
| 行をコピー | `Option + Shift + ↓` | `Alt + Shift + ↓` |
| 行を移動 | `Option + ↑/↓` | `Alt + ↑/↓` |

### その他

| 操作 | Mac | Windows |
| ---- | --- | ------- |
| ターミナルを開く | `Ctrl + `` | `Ctrl + `` |
| コマンドパレット | `Cmd + Shift + P` | `Ctrl + Shift + P` |
| 全検索 | `Cmd + Shift + F` | `Ctrl + Shift + F` |
