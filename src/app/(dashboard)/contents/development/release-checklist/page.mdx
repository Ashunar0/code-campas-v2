# リリース前チェックリスト

## この記事で学ぶこと

- デプロイ前の確認事項
- セキュリティチェック
- 最終確認

---

## デプロイ前チェックリスト

### 機能

```
- [ ] 全機能が動作する
- [ ] エラーハンドリングができている
- [ ] ローディング表示がある
- [ ] 空の状態でも表示が崩れない
```

### セキュリティ

```
- [ ] 環境変数が設定されている
- [ ] .env がコミットされていない
- [ ] Firebase セキュリティルールが設定されている
- [ ] 認証ガードが機能している
```

### UI/UX

```
- [ ] レスポンシブ対応
- [ ] 主要ブラウザで確認
- [ ] ファビコンが設定されている
- [ ] ページタイトルが適切
```

### ドキュメント

```
- [ ] README が充実している
- [ ] 使い方が書かれている
- [ ] スクリーンショットがある
```

---

## 最終確認

```
- [ ] デプロイ後に本番URLで動作確認
- [ ] チームメンバー全員で確認
- [ ] メンターにレビューを依頼
```

---

## まとめ

この記事では、リリース前のチェックリストを学びました。

- **機能・セキュリティ・UI/UX・ドキュメント**を確認する
- デプロイ後は **本番URL** で動作確認を行う
- チーム全員とメンターで **最終レビュー**をする
