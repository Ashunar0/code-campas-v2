## はじめに

プログラミングで「やりたいことを実現する」ために欠かせないのが、**条件分岐（if 文）** と**繰り返し処理（for 文）** です。

たとえば、「ある条件のときだけメッセージを表示したい」とか、「同じ処理を 10 回繰り返したい」と思ったことはありませんか？  
人間なら当たり前にやってる判断や繰り返しも、コンピュータにやらせるにはちゃんと命令を書く必要があります。

- **if 文**：条件によって処理を分ける（「もし〜なら、こうする」）
- **for 文**：同じ処理を何度も繰り返す（「これを 10 回繰り返す」）

この記事では、これらの構文を実際の例を通して、初心者の方にも分かりやすく解説していきます。変数の知識があれば、すぐに使えるようになります！

## if 文：条件分岐

### if 文とは？

if 文は、<strong>「条件が正しいかどうかによって、実行する処理を変える」</strong>ための構文です。日常生活でも「もし雨が降ったら傘を持っていく」という判断をしますよね。if 文は、そんな判断をプログラムで表現する方法です。

### if 文の基本的な書き方

```js
if (条件式) {
  // 条件が正しい（true）ときに実行される処理
} else {
  // 条件が正しくない（false）ときに実行される処理
}
```

### 実際の例

#### 例 1：年齢による入場判定

```js
let age = 18;

if (age >= 18) {
  console.log("大人料金で入場できます");
} else {
  console.log("子供料金で入場できます");
}
```

この例では：

- `age >= 18` が条件式（18 歳以上かどうか）
- 条件が正しければ「大人料金」のメッセージ
- 条件が正しくなければ「子供料金」のメッセージ

#### 例 2：テストの点数による評価

```js
let score = 85;

if (score >= 90) {
  console.log("優秀です！");
} else if (score >= 80) {
  console.log("よくできました！");
} else if (score >= 70) {
  console.log("まずまずです");
} else {
  console.log("もう少し頑張りましょう");
}
```

`else if`を使うと、複数の条件を順番にチェックできます。この例では 85 点なので「よくできました！」が出力されます。

### よく使う比較演算子

if 文で使う条件式には、以下のような比較演算子がよく使われます：

```js
let a = 10;
let b = 5;

a > b; // aはbより大きい
a < b; // aはbより小さい
a >= b; // aはb以上
a <= b; // aはb以下
a === b; // aとbは等しい
a !== b; // aとbは等しくない
```

## for 文：繰り返し

### for 文とは

for 文は、<strong>「同じ処理を指定した回数だけ繰り返す」</strong>ための構文です。例えば「1 から 10 まで順番に表示する」「配列の要素を全て処理する」といった場面で使います。

### for 文の基本的な書き方

```js
for (初期化; 条件; 更新) {
  // 繰り返し実行される処理
}
```

- **初期化**：繰り返しの準備（例：カウンターを 0 にする）
- **条件**：繰り返しを続けるかどうかの判断
- **更新**：繰り返しのたびに実行される処理（例：カウンターを 1 増やす）

### 実際の例で理解しよう

#### 例 1：1 から 5 まで数える

```js
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

実行結果：

```sh
1
2
3
4
5
```

この例では：

- `let i = 1`：カウンター（i）を 1 で初期化
- `i <= 5`：i が 5 以下の間繰り返す
- `i++`：繰り返しのたびに i を 1 増やす

#### 例 2：配列の要素を全て表示

```js
let fruits = ["りんご", "バナナ", "オレンジ", "ぶどう"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
```

実行結果：

```sh
りんご
バナナ
オレンジ
ぶどう
```

`fruits.length`は配列の要素数を表します。この例では 4 なので、0 から 3 まで（4 回）繰り返します。

#### 例 3：合計を計算する

```js
let numbers = [10, 20, 30, 40, 50];
let sum = 0;

for (let i = 0; i < numbers.length; i++) {
  sum = sum + numbers[i];
}

console.log("合計は: " + sum);
```

実行結果：

```sh
合計は: 150
```

## 実践的な例：FizzBuzz に挑戦しよう！

if 文と for 文を組み合わせた典型的な例として、<strong>FizzBuzz</strong>という問題に挑戦してみましょう。これはプログラミングの基礎を学ぶ上で、とても有名な問題です。

### FizzBuzz とは？

1 から 100 までの数字を順番に表示するプログラムを作ります。ただし：

- 3 の倍数のときは「Fizz」と表示
- 5 の倍数のときは「Buzz」と表示
- 3 と 5 の両方の倍数（15 の倍数）のときは「FizzBuzz」と表示
- それ以外の数字は、そのまま数字を表示

### 基本的な FizzBuzz

```js
for (let i = 1; i <= 20; i++) {
  if (i % 3 === 0 && i % 5 === 0) {
    console.log("FizzBuzz");
  } else if (i % 3 === 0) {
    console.log("Fizz");
  } else if (i % 5 === 0) {
    console.log("Buzz");
  } else {
    console.log(i);
  }
}
```

実行結果：

```sh
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
```

### コードの解説

1. **for 文**：1 から 20 まで繰り返す
2. **条件の順番**：重要なのは条件の順番です！

   - 最初に「3 と 5 の両方の倍数」をチェック
   - 次に「3 の倍数」をチェック
   - 次に「5 の倍数」をチェック
   - 最後に「それ以外」を処理

3. **剰余演算子（%）**：`i % 3 === 0`は「i を 3 で割った余りが 0」つまり「i が 3 の倍数」を意味します

### より効率的な書き方

条件を少し工夫すると、より簡潔に書けます：

```js
for (let i = 1; i <= 20; i++) {
  let output = "";

  if (i % 3 === 0) output += "Fizz";
  if (i % 5 === 0) output += "Buzz";

  console.log(output || i);
}
```

この書き方では：

- 空の文字列から始めて、条件に応じて文字を追加
- `output || i`は「output が空文字なら i を表示」という意味

### なぜ FizzBuzz が重要なのか？

FizzBuzz は以下の理由で、プログラミング学習に最適です：

- **if 文と for 文の組み合わせ**を実践できる
- **条件の優先順位**を理解できる
- **剰余演算子**の使い方を学べる
- **文字列の操作**も体験できる

実際のプログラミングでは、このような条件分岐と繰り返し処理の組み合わせが頻繁に使われます。FizzBuzz を理解できれば、多くの実用的なプログラムを作れるようになりますよ！

## よくある間違いと注意点

### if 文でよくある間違い

```js
// ❌ 間違い：= は代入演算子
if ((age = 18)) {
  console.log("18歳です");
}

// ✅ 正しい：=== は比較演算子
if (age === 18) {
  console.log("18歳です");
}
```

### for 文でよくある間違い

```js
// ❌ 間違い：無限ループになる可能性
for (let i = 0; i < 10; i--) {
  console.log(i);
}

// ✅ 正しい：カウンターを増やす
for (let i = 0; i < 10; i++) {
  console.log(i);
}
```

## まとめ

この記事では、JavaScript の if 文と for 文について学びました：

### if 文

- **目的**：条件によって処理を分ける
- **基本構文**：`if (条件) { 処理 } else { 処理 }`
- **応用**：`else if`で複数条件に対応

### for 文

- **目的**：同じ処理を繰り返す
- **基本構文**：`for (初期化; 条件; 更新) { 処理 }`

これらの構文は、プログラミングの基本中の基本です。実際にコードを書いて試してみることで、より深く理解できるようになります。次回は、これらの知識を使って、より実践的なプログラムを作ってみましょう！
