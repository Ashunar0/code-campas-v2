## よくある質問

### Q: React Hook Form は使うべき？

**A**: 複雑なフォームを多く作るなら使うと便利です。
シンプルなフォームなら手動バリデーションで十分です。

### Q: shadcn/ui と Tailwind どっちを使う？

**A**: 両方使います。
Tailwind は CSS の書き方、shadcn/ui はコンポーネント集です。

### Q: localStorage は実務で使う？

**A**: 簡易的なデータ保存には使います。
重要なデータは Firebase などのバックエンドに保存します。

### Q: コンポーネントはどこまで分割する？

**A**: 「1 つの役割」を意識します。
迷ったらまず分割せずに書いて、あとからリファクタリングしましょう。

### Q: テストは書くべき？

**A**: MVP 開発では必須ではありません。
しかし、チーム開発や長期運用する場合はテストが役立ちます。
