## ターミナルとは？

**ターミナル**は、コンピュータに「文字（コマンド）」で命令を出すためのツールです。

普段はマウスやトラックパッドでクリックして操作しますよね。
ターミナルでは、それを**文字を入力して**行います。

例えば：
- 「このフォルダを開いて」→ `cd フォルダ名`
- 「ファイルの一覧を見せて」→ `ls`
- 「新しいフォルダを作って」→ `mkdir フォルダ名`

最初は慣れないかもしれませんが、開発では毎日使うので、少しずつ慣れていきましょう！

## ターミナルの開き方

### Mac の場合

**方法 1: Spotlight 検索**
1. `Cmd + Space` で Spotlight を開く
2. 「ターミナル」と入力
3. Enter キーで起動

**方法 2: Finder から**
1. Finder を開く
2. アプリケーション → ユーティリティ → ターミナル

{/* 
📸 画像メモ: Mac の Spotlight でターミナルを検索している画面
*/}

### Windows の場合

Windows では「コマンドプロンプト」または「PowerShell」を使います。

**方法 1: 検索から**
1. スタートメニューの検索バーに「cmd」または「PowerShell」と入力
2. クリックして起動

**方法 2: Windows + R**
1. `Windows + R` を押す
2. 「cmd」と入力して Enter

## 基本コマンド

ここでは、よく使う基本的なコマンドを紹介します。

### pwd（現在地を確認）

```bash
pwd
```

**p**rint **w**orking **d**irectory の略。
「今、自分がどのフォルダにいるか」を表示します。

```
/Users/tanaka/Desktop
```

📸 画像メモ: pwd コマンドの実行結果
*/}

### ls（ファイル一覧を表示）

```bash
ls
```

**l**i**s**t の略。
現在のフォルダにあるファイルやフォルダの一覧を表示します。

```
Documents  Downloads  Desktop  Pictures
```

{/* 
📸 画像メモ: ls コマンドの実行結果
*/}

### cd（フォルダを移動）

```bash
cd フォルダ名
```

**c**hange **d**irectory の略。
指定したフォルダに移動します。

```bash
# Desktop フォルダに移動
cd Desktop

# 1つ上のフォルダに移動
cd ..

# ホームフォルダに戻る
cd ~
```

{/* 
📸 画像メモ: cd コマンドでフォルダを移動している様子
- cd Desktop を実行
- pwd で確認すると /Users/tanaka/Desktop になっている
*/}

### mkdir（フォルダを作成）

```bash
mkdir フォルダ名
```

**m**a**k**e **dir**ectory の略。
新しいフォルダを作成します。

```bash
# projects フォルダを作成
mkdir projects

# 中身を確認
ls
```

### touch（ファイルを作成）

```bash
touch ファイル名
```

新しい空のファイルを作成します。

```bash
# index.html を作成
touch index.html

# 中身を確認
ls
```

> ⚠️ **Windows の場合**
> 
> Windows のコマンドプロンプトでは `touch` コマンドは使えません。
> 代わりに `echo. > ファイル名` を使うか、Git Bash を使いましょう。

## VSCode の統合ターミナル

実は、VSCode には**ターミナルが内蔵**されています。
開発中は、VSCode の統合ターミナルを使うのが便利です。

### 開き方

`Ctrl + `` （バッククォート）を押すと、画面下部にターミナルが開きます。

### 統合ターミナルの利点

- VSCode を開きながら、ターミナルを使える
- 現在開いているフォルダで自動的にターミナルが開く
- 複数のターミナルを開ける

## パス

コマンドを使う上で「**パス（path）**」の理解が大切です。

### 絶対パス

コンピュータのルート（一番上）からの完全な住所。

```
/Users/tanaka/Desktop/projects/my-app
```

### 相対パス

現在地からの相対的な位置。

```bash
# 現在地が /Users/tanaka/Desktop の場合
cd projects/my-app  # 相対パスで移動
```

### 特殊な表記

| 表記 | 意味 |
| ---- | ---- |
| `.` | 現在のフォルダ |
| `..` | 1つ上のフォルダ |
| `~` | ホームフォルダ |
| `/` | ルート（一番上） |

## 練習してみよう

以下の手順を試してみましょう：

1. ターミナルを開く
2. `pwd` で現在地を確認
3. `cd Desktop` でデスクトップに移動
4. `mkdir test-folder` でフォルダを作成
5. `cd test-folder` でその中に移動
6. `touch hello.txt` でファイルを作成
7. `ls` でファイルを確認
8. `cd ..` で1つ上に戻る
