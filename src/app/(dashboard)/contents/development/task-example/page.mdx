# タスク分割の実例

## この記事で学ぶこと

- Todo アプリのタスク分割例
- 各タスクの詳細

---

## Todo アプリのタスク分割

```
Week 1:
├── #1 プロジェクト作成・環境構築
├── #2 Firebase プロジェクト作成・設定
├── #3 ログイン機能実装
└── #4 ユーザー登録機能実装

Week 2:
├── #5 Todo 一覧取得・表示
├── #6 Todo 追加機能
├── #7 Todo 完了/未完了切り替え
├── #8 Todo 編集機能
└── #9 Todo 削除機能

Week 3:
├── #10 検索機能
├── #11 フィルター機能
├── #12 スタイリング
└── #13 デプロイ
```

---

## 各タスクを Issue 化

### #1 プロジェクト作成・環境構築

```
- Vite + React でプロジェクト作成
- Tailwind CSS を設定
- ESLint / Prettier を設定
- GitHub リポジトリ作成
```

### #5 Todo 一覧取得・表示

```
- Firestore から todos を取得
- TodoList コンポーネントを作成
- TodoItem コンポーネントを作成
- ローディング表示を実装
```

---

## 役割分担の例（3人チーム）

```
Aさん（リーダー）:
├── #1 環境構築
├── #2 Firebase 設定
└── #12 スタイリング

Bさん:
├── #3 ログイン
├── #4 登録
└── #5 一覧表示

Cさん:
├── #6 追加
├── #7 完了切り替え
├── #8 編集
└── #9 削除
```

## まとめ

この記事では、Todo アプリを例にタスク分割の実例を学びました。

- 週ごとにタスクを整理し、**段階的に実装**する
- 各タスクを **Issue 化**して管理する
- チームでは**役割分担**を明確にする
