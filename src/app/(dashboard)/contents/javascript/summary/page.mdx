## React でどう活かされるか

JavaScript で学んだ内容は、React でそのまま使えます。

### アロー関数 → コンポーネント

```jsx
// React のコンポーネント = アロー関数
const Greeting = ({ name }) => {
  return <h1>こんにちは、{name}さん！</h1>;
};
```

### map → リスト表示

```jsx
const users = ["田中", "佐藤", "鈴木"];

return (
  <ul>
    {users.map((user) => (
      <li key={user}>{user}</li>
    ))}
  </ul>
);
```

### 分割代入 → Props

```jsx
// props を分割代入で受け取る
const UserCard = ({ name, age, email }) => {
  return (
    <div>
      <h2>{name}</h2>
      <p>{age}歳</p>
      <p>{email}</p>
    </div>
  );
};
```

### スプレッド構文 → State 更新

```jsx
const [user, setUser] = useState({ name: "", age: 0 });

// スプレッド構文で一部だけ更新
setUser({ ...user, name: "田中" });
```

### async/await → データ取得

```jsx
useEffect(() => {
  const fetchData = async () => {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    setData(data);
  };
  fetchData();
}, []);
```

### filter → 削除機能

```jsx
const deleteTodo = (id) => {
  setTodos(todos.filter((todo) => todo.id !== id));
};
```
