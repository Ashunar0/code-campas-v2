## この章について

この章では、Web ページの見た目を作るための **HTML** と **CSS** を学びます。

- **HTML**：ページの「構造」を作る（骨組み）
- **CSS**：ページの「見た目」を整える（装飾）

この 2 つは、Web 開発の基礎中の基礎です。

{/*
📸 画像メモ: HTML と CSS の役割を示す図
- 左: HTML = 骨組み（構造）
- 右: CSS = 装飾（見た目）
- 同じ HTML でも CSS を変えると見た目が変わる例
*/}

## なぜ HTML / CSS を学ぶのか

「React を使うのに、HTML / CSS って必要なの？」と思うかもしれません。

答えは **「はい、必要です」**。

React では **JSX** という形式でコードを書きますが、これはほぼ HTML と同じです。
また、スタイリングには CSS（や Tailwind CSS）を使います。

つまり、HTML / CSS を知らないと React も書けないんです。

ただし、**完璧に覚える必要はありません**。
React で必要になる最低限の知識を、この章でサクッと身につけましょう。

## 学習の進め方

### この章は「自習」がメイン

HTML / CSS はゼミで詳しく扱いません。
**自分で教材を読み進めながら学習**してください。

わからないことがあれば、メンターに質問して OK です！

### 完璧を目指さない

HTML のタグや CSS のプロパティは非常に多いです。
全部覚えようとすると大変なので、**よく使うものだけ** 覚えましょう。

わからないことがあれば、その都度調べれば大丈夫です。
プロのエンジニアも毎日検索しています。

### 手を動かすことが大切

教材を読むだけでなく、**実際にコードを書いて試す**ことが大切です。
特にハンズオン（01-08）では、自分でポートフォリオサイトを作ります。
